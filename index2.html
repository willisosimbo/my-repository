How does writing a CSS differ from writing HTML

As we write css these are the types of questions we ask ourselves

  • Should we layout the line in a single or double column
  • Should we use a different color for the font header
  • How should the very same content appear in a desktop vs mobile device

All this questions above deal with esthetic and function consideration

Lets consider the types of questions we might ask ourselves as we write HTML

  • Does the order of items within s list matter? Should it be numbered?
  • Should we wrap a list of links inside a navigation tag?
  • Is this the most important header in the HTML document

The last few questions deal with structure, hierachy and meaning. These are perspective of content layer in HTML

When we write CSS we focus on esthetic and function consideration. When we write HTML we focus on structure,hierachy and meaning
